Counting functions

Let A={1,2,3,…,m} , B={1,2,3,…,n}

How many functions are there?

nm

Clearly we can construct this function in nm ways.

 

[Prop.] If A, B are finite sets, then the number of functions f:A↦B is |B||A|

[Prop.] If |A|=|B|=n, then the number of bijective f:A↦B is n!

Note: if |A|≠|B|, no bijective function can be built.

 

Problem: How many ways can 10 books be arranged onto a shelf?

Sol: 10!

10 9 8 7 ... 1

We can view this as a bijection -- f:{10slots}↦{10books}

 

[Prop.] Let |A|=m,|B|=n. The number of injective f:A↦B is

  • if m>n, 0
  • if m≤n, n!(n−m)! injective functions

Note: 0!=1 by definition

Total=n(n−1)(n−2)…(n−(m−1))=n(n−1)…(n−(m−1))(n−m)…1(n−m)!=n!(n−m)!

 

Problem. Given 10 books, in how many ways can we arrange 5 of them on a shelf?

Sol: 10p5=10!5!

10 9 8 7 6

Each arrangement is an injective function -- f:{5slots}↦{10books}

 

Note: The following numbers are the same

  1. Number of f:A↦B injective
  2. Ways to arrange |A| objects selected from a set of size |B|
  3. Number of oredered subsets of size |A| from a set of size |B|

 

Surjectice functions

How many f:A↦B are surjective?

  2. If |A| = |B|, surjectice⇔bijective so n!

[Prop.] Let |A|=n. The number of subsets of A with k elements (k≤n) is n!k!(n−k)!

Def

(nk)=n!k!(n−k)!

 

[Prop.] Let |X|=n. The number of subsets of X with k elements is n!k!(n−k)!

Method of Double-counting: To prove m=n, find a set A such that |A|=m, |A|=n

Method of Bijection: To count A, find a bijective function f:A↦B, count B instead (|A|=|B|)

[Prop.]

∑k=0n(nk)2=(2nn)

Proof:

Let X={1,2,…,2n}, we will count the number of n-element subsets of X in 2 ways.

Method 1:

Choose n-element subsets from {1,2,...,2n}

So (2nn)

Method 2:

Let us divide it into mutually exclusive cases.

[number of n-element subsets] = ∑k=0n[number of n-element subsets that have k elements in {1,2,...,n}]

= ∑k=0n[# of k-element subsets of {1,..,n}][# of (n-k)-element subsets of {n+1,...,2n}]

= ∑k=0n(nk)(nn−k)

= ∑k=0n(nk)(nk)

We proved RHS and LHS actually count the same thing.

 

Pascals Triangle

[Prop.]

(nk)=(n−1k)+(n−1k−1) for all n,k≥1

  • Algebraic Proof

    Omit (I am too lazy to type them out :D)

  • Combinatorial Proof:

    Count A={1,2,…,n} in 2 ways.

    Let us divide it into mutually exclusive cases.

    Case 1: subsets contained in B={1,…,n−1}

    Then we have (n−1k) options.

    Case 2: subsets not contained in B

    We must include n since we are counting the number of subsets that are not contained in B

    Pick k−1 elements from B, then (n−1k−1) options.

    In total, (n−1k)+(n−1k−1)

 

We can show (nk) actually gives Pascal's triangle by induction proof using the proposition proved above.

[Prop.]

2n=∑k=0n(nk)

Proof

  • Binomial theorem (omit)
  • Combinatorial Proof

Let X be a set with |X|=n. Count P(x)

Method 1:

When determining a subset we have 2 options for each element -- include or exclude

2 2 2 2 2... 2

X=x1,x2,x3,...xn

|P(x)|=2n (It's 2 to the power of n since there are 2 options -- include or exclude)

Method 2:

Divide into mutually exclusive cases

[|P(x)|] = [# subsets with 1 element]+[# subsets with 1 element]+...+[# subsets with n elements]

= ∑k=0n[# of subsets with k elements]

= ∑k=0n(nk)

Both Method 1 and Method 2 count |P(x)|. ◼
